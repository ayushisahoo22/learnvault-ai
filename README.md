# 🧠 LearnVault AI

🚀 An AI-powered learning workspace that automatically categorizes conversations and helps users revisit knowledge efficiently.

## 🎯 Problem Statement

Students often discuss multiple topics with AI tools and later struggle to find where they learned a concept. Important information gets buried inside long chat histories.
LearnVault solves this by organizing chats automatically and making learning easier to revisit.

## ✨ Features

* 🤖 AI-based automatic topic categorization
* 📂 Topic-wise chat organization
* 🔍 Search previous conversations
* 📝 Generate concise crux/notes on demand
* 📌 Pin important concepts
* 💬 Chat history management
* 🔐 Authentication system
